Station banner

Station BannerLine 1Line 2Line 3

How this component works

This station banner will be automatically styled as long as a matching station is found in the stations.json file

The page title field (h1 text contnet) will be fed to the stations.js which fetches the stations.json data file to find the matching station to lookup the station data to populate the following elements dynamically

  1. Station line icons (if applicable)
  2. Train icon for train stations (if applicable)
  3. Bus icon for bus stations (if applicable)

How to manipulate this component

The icons generated on the left hand side to the h1 text content are dependent on boolean the values of the following properties of the matching station in the stations.json file station_type.line_1 station_type.line_2 station_type.line_3 station_type.line_4 station_type.bus

Quicklink card

Condition
Park & Ride

IF station_url.pnr value exists in stations.json file
