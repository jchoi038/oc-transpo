Hyperlinking guide

				

					
						

							Topic
							Details
						

					
					
						

							Link text
							
								

									The text contained within a hyperlink should be
									sufficiently specific to allow a user hearing
									the link by itself to understand where it is
									heading.
								

								

									Meaningful link text should not be overly
									general and should clearly describe the content
									to be found or action to be performed by the
									link. (Ensure link text is meaningful within
										context - webaccessibility.com)
								

									
						

						

							Linking to a page within a PDF hosted online:
							
								

									To link to a specific page, append the following
									code to the URL: #page=[pagenumber].
								

								

									Example:
									https://domain.com/manuals/72041M.pdf#page=41
									links to page 41 of the PDF.
								

									
						

						

							Adding the PDF icon to a hyperlink
							
						

						

							Linking to an external site
							
								

									To link to an external site, add the following
									class to the anchor tag:
								

								
class="link-new-window"

								

									This will add the title “Opens external website
									in a new window/tab.” And append the following
									icon
