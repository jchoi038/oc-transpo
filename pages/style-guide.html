Colours

				
Primary, secondary colours, hues & tints

			

			

				
Colour value meanings

				
50: Contrast of low 1.0s. Used as subtle backgrounds for banners & section dividers

				
100: Contrast of mid 1.0s

				
50-200s: light bg fills for dark text, Hover shadows

				
200-300s: Graphics, Hover shadows

				
50-400s: non cruicial / non interacting ui elements (dividers, backgrounds)

				
500s: Contrast of 3.0+. crucial ui components: ui borders, icons, text 18 pt / 14 pt bold

				
600s: Contrast of 4.5+. Prominent bg fills for light text, normal text

				
700s: Button hover state fills

				
800s: dark text on white or light bg (up to 200s)

The ToC element will be automatically populated with the text content of all h2 tags in the page
				and the links to the sections

			
for the page to be read correctly, follow the guide below:

			
    
				
  1. Add <div class="toc sticky"></div> to the top of your page
				
    2. 
				
  2. 
				
  3. Use a section tag to divide up main sections of page you would like to be listed in the
					ToC.
    4. 
				
  4. For each section, there should be one h2 element
    5. 
				
  5. For each section, add a unique id.
    6. 
				
  6. The text content of the h2 tag will be grabbed and put as the text contnet for the ToC
					links
    7. 
				
				
    Example:
    
				
    	<div class="toc sticky"></div>
	<section id="line1"> //id used in href for ToC links
		<h2>Line 1</h2> //text content from h2 grabbed and put in ToC
	</section>
	<section id="line2"> //id used in href for ToC links
		<h2>Line 2</h2> //text content from h2 grabbed and put in ToC
	</section>
	
    
				
				
    For the above example, the contents of the ToC will be "Line 1" and "Line 2" with hrefs of
					#line1 and #line2, respectfully
    
				
    
					
    To ensure your id is unique, it is recommended you put a prefix a shortform of the page name to the
						id.

						line1 used in the example
