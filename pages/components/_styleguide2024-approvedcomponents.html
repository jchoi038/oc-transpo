Icon naming convention guide

Format:

 
                    [A]
                    _
                    [B]
                    .svg
[A]

Category Name:

  • Pick a category from the following options: 'line' / 'route' / 'transport' / 'semantic' / 'accessibility' / 'station' / 'resource' / 'destination' / 'interactive' / 'contact' / 'detour' / 'map' / 'tags' / 'service' / 'msc' / 'link-icon'
  • Use lowercase letters only.
    • [B]

    Icon Name:

  • Name the icon using kebab-case format.
  • Correct: happy-goat
  • Incorrect: happyGoat, happy_goat
  • Use lowercase letters only.
    • _

    Linking [A] and [B]

  • Combine the category name and icon name with an underscore _.
  • Example:'contact_email.svg'
    • .svg

    Upload format

  • upload your icon in '.svg' format

    • Upload directory

    All icons are to be saved in 

    Assets/Uploads/icons

    Usage guide

    Customizing text:
    • While the text within the tags can be customized to fit your needs, the semantic meaning conveyed by the color should remain unchanged.
    • Choose text that clearly represents the status in a concise and understandable manner.

    When to use

    To represent the status of an element on a page.

    How to use
    • Within card components: Use to display the status of an item or entity within a card.
    • Within table components: Use to display the status of items within table rows.

    Keyword tag

    The Keyword tag component is used to label and organize data within card components or other UI elements. It helps in categorizing information without implying a status or urgency.

                    <p class="tag tag-type__keyword">Keyword</p>

    Projects

    Usage guide

    When to use:

    The Keyword tag component is designed for simple, neutral labeling without any additional styles. It is ideal for use within card components or other UI elements where a simple text label is sufficient.

    How to use:

    Within card components: Place above card title.

    To avoid:

    Refrain from making keyword more than one word.
