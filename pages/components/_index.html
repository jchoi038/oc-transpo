Usage guide

Customizing text:
  • While the text within the tags can be customized to fit your needs, the semantic meaning conveyed by the color should remain unchanged.
  • Choose text that clearly represents the status in a concise and understandable manner.

When to use

To represent the status of an element on a page.

How to use
  • Within card components: Use to display the status of an item or entity within a card.
  • Within table components: Use to display the status of items within table rows.

Keyword tag

The Keyword tag component is used to label and organize data within card components or other UI elements. It helps in categorizing information without implying a status or urgency.

Keyword

                <p class="tag tag-type__keyword">Keyword</p>

Examples

Projects

Feature

Usage guide

When to use:

The Keyword tag component is designed for simple, neutral labeling without any additional styles. It is ideal for use within card components or other UI elements where a simple text label is sufficient.

How to use:

Within card components: Place above card title.

To avoid:

Refrain from making keyword more than one word.

Semantic note usage guide

Warning note title

Warning notes are to be used as generic warning messages like service disruptions, important reminders.

Tone
  • Important
  • non-urgent

When to use
  • Generic warning messages / non-urgent, important reminders.

Specific use case:
  • Service disruptions

Warning note title examples
  • Warning
  • Reminder
  • Dates adjusted for Canada Day
  • Tap your card, not your wallet
  • Line 1 planned maintence

Urgent note title

Lorem ipsum dolor sit amet consectetur. Libero rhoncus mi gravida eget magna suscipit lobortis lorem enim. Amet in eget vitae phasellus condimentum diam iaculis.

Urgent notes are to be reserved for messages of high importance and urgency, considered a level higher in importance compared to warning notes. Do not use them for regular service disruptions; instead, use the warning note.

Tone
  • Urgent

When to use
  • For high importance messages.

Specific use cases
  • Pandemic-related government mandates (e.g., COVID-19 mask requirement, social distancing, etc.)
  • Current urgent issues (e.g., weather, terrorism) that require immediate attention

Urgent note title examples
  • COVID-19 (or other future pandemic names)
  • Important
