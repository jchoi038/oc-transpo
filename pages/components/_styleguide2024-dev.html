Stations

This style guide component uses a script to grab data from stations.json file to auto-generate the list of stations and place them in their respective sections

List of auto generated elements:

  • Prepended h2 element with applicable Train/bus icon + Line icon + Title of section

  • All stations for that section with class="select-option"

  • Applicable bus/line/pnr icons for each individual station

How to use these components

1. Add the master stylesheet to the top of your content

<link href="/images/files/dist/custom-style.css" rel="stylesheet" />

2. Add the following script file to the bottom of your content

<script src="/images/files/app/stations.js"></script>

3. Copy and paste any of the section elements below to auto generate the stations belonging to their respective section

  • <section id="line1Stns"></section>
  • <section id="future_line1Stns"></section>
  • <section id="line2Stns"></section>
  • <section id="line4Stns"></section>
  • <section id="line3Stns"></section>
  • <section id="future_line3Stns"></section>
  • <section id="busStns"></section>
