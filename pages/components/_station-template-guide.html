Overview:

These instructions will allow styling of the page h1 and automatically add the corret set of icons for the station (lines icons (if applicable) , train icon (if applicable), and bus icon (if applicable))

How to use:

1. Change The title field to a station name (This is important because the title will be set to the text content of the h1 which will be used in looking up data files with the js script)

2. Add the following style sheet link to the top of the body:

<link href="/images/files/dist/custom-style.css" rel="stylesheet" />

3. Add the following js script to the end of the body:  

<script src="/images/files/app/stations-template.js"></script><script src="/images/files/app/stations-template.js"></script>

This component only works on wide page templates

Quick links (Auto generated)

<div id="quicklinks-container" class="grid grid-5-4-2-2"></div>

All quick link cards are all auto generated depending on station data & page sections

  • Schedules card generated IF station schedule link exists in stations.json

  • Station layout card generated IF section with <section>id="section_station-layout" </section>

  • Amenities card generated IF section with <section>id="amenities" </section>

  • Boarding locations card generated IF section with <section>id="stops" </section> exists AND IF Train or Bus stops exist in stopRecipes.json

  • 360-Deg tour card generated IF section with <section>id="360" </section> exists AND IF 360 url exist in stations.json

  • Waht's nearby card generated IF section with <section>id="nearby" </section> exists on page

  • Parking card generated IF section with <section>id="360" </section> exists AND IF 360 url exist in stations.json

Station hero banner (Auto generated)

        

            
        

        
            
How to use:

        
        
Add the following div to the top of the body (below the styles link)

        
        <div id="station-hero-banner" class="hero-banner">
        <img id="station-hero-img">
        </div>

        
Hero banner image & contents of station information card is dynamically populated depending on the station
        

        
the src of the hero banner background image is dynamically populated depending on the station

        
This src is from JSON file: data/stations.JSON any changes can be manipulated by changing the data in this
            JSON
            file
        

        

            This component only works on wide page templates
        

        

            Some station do not have hero banners yet assigned.
